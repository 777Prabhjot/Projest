Adding a Project
Navigate to the "Projest" platform. Click on the "Projects" dropdown menu in the navigation bar. Select the "Add Project" option. Fill in the form with your project details, including the project title, description, user Github link, and project Github link. Select the tags that best describe your project. Click the "Submit" button to send the form data to the real-time Firebase database.
Viewing All Projects
Navigate to the "Projest" platform. Click on the "Projects" dropdown menu in the navigation bar. Select the "View All Projects" option. All the projects that have been submitted to the platform will be displayed. You can search for a specific project by its title or tag name using the search bar.
Contributing to a Project
Navigate to the "Projest" platform. Select the "Contribute" page from the navigation bar. Find a project that you would like to contribute to. Follow the project's Github link to access its repository. Make your contributions to the project. Submit your changes via a pull request.
